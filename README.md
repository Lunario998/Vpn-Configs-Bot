# Vpn-Configs-Bot

Простой Telegram-бот, который выдает рабочие VPN-конфиги.

## Быстрый запуск

1. Установи Python 3.12+.
2. Открой папку проекта.
3. Создай виртуальное окружение:

```bash
python3 -m venv .venv
```

4. Активируй окружение:

```bash
source .venv/bin/activate
```

5. Установи зависимости:

```bash
pip install -r requirements.txt
```

6. Создай `.env` на основе `.env.example` и заполни нужные поля.

7. Запусти бота:

```bash
python main.py
```
